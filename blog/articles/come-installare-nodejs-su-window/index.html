Come installare node.js su window 10

Installare Node JS sulla macchina è un'operazione molto semplice, basta recarsi presso il sito ufficiale nodejs.org e cliccare sotto la scritta Download for Windows (x64) dove riportata l'ultima versione stabile la LTS version, attualmente la 14.15.4

Una volta scaricato e avviato il wizard di installazione, basterà leggere e in una manciata di "Next"avrete installato Nodejs sulla vostra macchina non resta altro che vedere se tutto è andato a buon fine,aprite un Prompt dei Comandi Windows ( basterà aprire window e cercare CMD ) e lanciateil comando "node", se tutto è andato a buon fine dovreste vedere un messaggio di Welcome segiuto dallaversione

Ben fatto ora potrai esegiure codice javascript senza usare alcun browser !
